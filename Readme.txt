Read Me!

For unity plug-in:
1.REMINDER! The location of AndroidManifest.xml and PhotoCropper.jar must be placed in the Asset/Plugins/Android path. Please move them out of the folder. Otherwise, the plug-in might not run.
2.The source code for the Android section is in source_code.zip. I use Android Studio to export the .arr, and pick the classes.jar from the .aar.(REMINDER: DO NOT pick the classes.jar from the lib dictionary) If you do not know how to export .jar, a video (https://www.bilibili.com/video/BV1Qt4y1a7aW?from=search&seid=8004011218970423000&spm_id_from=333.337.0.0) is highly recommended. This video is only available in Chinese. Or you can check the PhotoCropper Android Studio Project.
3. This plugin works fine on Unity 2019.4.21f1c1. Unity 2019.4.21f1c1 and later versions should also work properly. It also works fine on most Android phones, including Huawei and Xiaomi.


对于这个插件，请注意以下几点：
1. AndroidManifest.xml and PhotoCropper.jar的位置必须位于Asset/Plugins/Android路径下，请把它们从文件夹中拖出来。否则，插件将无法运行。
2. 关于安卓部分的源码在source_code.zi文件夹中。我用 Android Studio来导出aar包，然后从aar包中取出了jar包（不是在libs里面的jar包）。如果你还不知道如何导出jar包，请参考https://www.bilibili.com/video/BV1Qt4y1a7aW?from=search&seid=8004011218970423000&spm_id_from=333.337.0.0 这个视频，非常非常非常非常非常有用！！
3. 这个插件在Unity 2019.4.21f1c1版本上可以正常使用。Unity 2019.4.21f1c1以上版本应该也可以使用。这个插件支持大多数安卓手机，包括华为和小米手机。
